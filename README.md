# 🧠 Quinfal_auction_bot — игровой бот для поиска задач и покупки предметов на аукционе

## 📌 Описание проекта

**Quinfal_auction_bot** — это автоматизированный игровой помощник, который:

- ищет **доску задач** в игровом мире,
- извлекает названия задач,
- автоматически переходит в **аукцион**,
- ищет нужные предметы,
- сравнивает цены,
- покупает предметы по заданным условиям,
- выводит статус работы в **прозрачном PyQt5‑интерфейсе** поверх игры.

Бот работает через `PyAutoGUI`, `OpenCV`, `PyDirectInput` и собственную базу изображений (`knowledge_base`).

---

## 🚀 Основные возможности

- Поиск доски задач по изображению  
- Распознавание элементов интерфейса через `locateCenterOnScreen`  
- Автоматический поиск предметов на аукционе  
- Покупка предметов при совпадении цены  
- Прозрачный PyQt5‑overlay с логами  
- Гибкая `knowledge_base` с изображениями кнопок и элементов  


---

## 🔧 Установка и запуск

pip install -r requirements.txt
